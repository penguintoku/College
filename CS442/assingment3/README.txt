PURPOSE:

	The purpose of this project is:
		1. Not to use Eclipse
		2. Make a program using a push method Observer pattern

	The code will display the full list of results if the names are in the list.
	If the names are not in the list, it will not display.

FILES:

	There is a total of 9 files for this project:

		build.xml, the file that will compile the code
				
		README.txt, the current file you are reading that has all the information

		Logger.java, the class that will display all message depending on the DEBUG_LEVEL

		Driver.java, the main class that will run the program

		Observer.java, the observer interface for the observer

		Subject.java, the subject interface for the subject

		Filter.java, the filter class that will see valid of the tag

		Parser.java, the parser to read in the user input

		Performance.java, the performance tab option

		Process.java, the process tab option

		Status.java, enum class for the status of the user

		TaskData.java, The controlling of the objects

		testData.txt, the file that will be acting as the reading the data live

		Users.java, the user tab option
